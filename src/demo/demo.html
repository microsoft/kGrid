Getting started

  1. "Add references to jQuery and jQueryUI"
  2. "Add references to kGrid js and css files"
  3. "Set your target element in your html file"
    <div id="basic"></div>
  4. "In your JavaScript file, initialize your element"
        var root = $('#basic');
    var listControlElement = $('<div style="position: relative;"></div>');
    root.append(listControlElement);
    listControlObject = new Microsoft.Office.Controls.ListControl(listControlElement[0]);
  5. "Add columns"
        listControlObject.addColumns([
        { data: 'Name', field: 'name', table: { width: 100 }, },
        { data: 'Age', field: 'age', table: { width: 100 }, },
    ]);
  6. "Add data" 
        listControlObject.rows([
        { name: 'test1', age: 25 },
        { name: 'test2', age: 23 },
        { name: 'test3', age: 12 },
        { name: 'test4', age: 42 },
        { name: 'test5', age: 26 },
        { name: 'test6', age: 36 },
    ]);

Basic Example

Right to Left edit

  1. Set property rtl to true; 
    listControlObject.rtl(true);
  2. update UI; 
    listControlObject.updateUI();

Stack View

  1. Set property viewType to Stack or Table ; 
    listControlObject.viewType(Microsoft.Office.Controls.ViewType.Stack);
  2. update UI; 
    listControlObject.updateUI();

Selection

  1. Set selectin mode, SingleRow, MultipleRows, Range or Cell, 
    listControlObject.selectionMode(selectionMode);
  2. invalidate; 
    listControlObject.invalidate();

Theme

  1. Create your theme object; 
            var theme = {
            'backgroundColor': '#ffffff',
            'hoverBackgroundColor': '#f4f4f4',
            'selectionBackgroundColor': '#cde6f7',
            ...
        }
  2. Set theme; 
    listControlObject.theme(thme);
